# Basic Skill

## When to use this skill

Use this skill as a reference when creating your first Agent Skill.

## Instructions

1. Read the user's request carefully
2. Identify the specific task they need help with
3. Provide a clear, step-by-step response

## Gotchas

- This is a template skill — replace all content with your own instructions
- Keep the SKILL.md body under 500 lines for optimal agent performance
