# Supported Languages

| Code | Language   | Formal Support |
|------|------------|----------------|
| en   | English    | Yes            |
| es   | Spanish    | Yes            |
| fr   | French     | Yes            |
| de   | German     | Yes            |
| ja   | Japanese   | Yes            |
| zh   | Chinese    | No             |
| ko   | Korean     | No             |
| pt   | Portuguese | No             |
| ar   | Arabic     | No             |
| hi   | Hindi      | No             |

## Adding a new language

1. Add the ISO 639-1 code and greeting to `scripts/greet.js`
2. Update this table
3. Run `skill validate` to verify
